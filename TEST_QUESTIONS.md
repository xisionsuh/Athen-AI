# 협업 모드 테스트 질문

각 협업 모드를 테스트하기 위한 예시 질문입니다.

## 1. Single 모드 (단일 AI)
**특징**: 간단한 질문, 특정 분야에 특화된 질문, 빠른 응답이 필요한 경우

**예시 질문:**
```
파이썬에서 리스트를 정렬하는 방법을 알려줘
```

**예상 동작**: ChatGPT 하나만 사용하여 빠르게 답변

---

## 2. Parallel 모드 (병렬 협업)
**특징**: 다양한 관점이 필요한 복잡한 질문, 여러 측면을 동시에 분석해야 하는 경우

**예시 질문:**
```
인공지능이 미래 사회에 미칠 긍정적 영향과 부정적 영향을 종합적으로 분석해줘
```

**예상 동작**: 여러 AI가 동시에 각자의 관점에서 답변하고, 총괄 AI가 종합

---

## 3. Sequential 모드 (순차 협업)
**특징**: 단계별로 나누어 처리해야 하는 복잡한 작업, 보고서 작성, 심층 연구

**예시 질문:**
```
2024년 AI 트렌드를 조사하고, 각 트렌드의 기술적 배경, 시장 영향, 향후 전망을 포함한 종합 보고서를 작성해줘
```

**예상 동작**: 각 AI가 순차적으로 작업을 이어받아 단계별로 처리

---

## 4. Debate 모드 (토론 모드)
**특징**: 논쟁적 주제, 서로 다른 입장이 필요한 주제, 균형잡힌 결론이 필요한 경우

**예시 질문:**
```
원자력 발전소 확대 vs 축소에 대해 각 AI의 입장을 듣고 싶어
```

**예상 동작**: 각 AI가 서로 다른 입장에서 토론하고, 총괄 AI가 균형잡힌 결론 도출

---

## 5. Voting 모드 (투표 모드)
**특징**: 여러 선택지 중 하나를 선택해야 하는 경우, 기술 스택 선택, 의사결정이 필요한 경우

**예시 질문:**
```
스타트업을 위한 최적의 프로그래밍 언어는 무엇인가? Python, JavaScript, Go 중에서 선택해줘
```

**예상 동작**: 각 AI가 의견과 선택을 제시하고, 총괄 AI가 투표를 집계하여 최종 결론 제시

---

## 테스트 순서 추천

1. **Single 모드** → 가장 빠르고 간단함
2. **Voting 모드** → 선택지가 명확하여 결과 확인이 쉬움
3. **Parallel 모드** → 여러 AI의 관점을 비교하기 좋음
4. **Debate 모드** → 토론 과정을 관찰하기 좋음
5. **Sequential 모드** → 단계별 진행을 확인하기 좋음

## 확인 방법

각 질문을 입력한 후:
- 서버 콘솔에서 `🎬 스트리밍 모드: [모드명]` 로그 확인
- 브라우저 콘솔에서 협업 모드 관련 로그 확인
- 응답의 메타데이터에서 `전략: [모드명]` 확인
- 각 모드별로 다른 진행 상황 로그 확인
